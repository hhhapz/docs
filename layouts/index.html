Database First

Generate code via database introspection for your tables, indexes, foreign keys, stored procedures and views.

Major DB Support

Most major relational databases are supported. XO supports PostgreSQL, MySQL, Oracle Database, SQL Server, and SQLite3.

Idiomatic Code

Xo automatically creates data structures, CRUD operations, and functions from your indexes and foreign keys that just make sense.

Custom Templates

Easily customise, configure, and tweak XO to your liking. If XO doesn't have a feature you need, easily create your own templates.

Easy to use

To get started with XO, run
xo schema [db url].
That's it! Your schema is used to
generate idiomatic code.

Really Fast

XO is fast. Both with generating your code, and during runtime. Simple queries that do just what you need make sure things run smoothly.
